guitool_ibeis
=============

|Pypi| |Downloads| |Codecov| |Travis| |Appveyor| 

Helpers for PyQt5 GUI development for the IBEIS project.

Depends on `vtool_ibeis` and `utool`.

Vision Tools - tools for computer vision. Part of the WildMe / IBEIS Project.
